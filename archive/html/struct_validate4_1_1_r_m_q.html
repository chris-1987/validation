Detailed Description

template<typename alphabet_type, typename alphabet_extension_type, typename size_type>
struct Validate4< alphabet_type, alphabet_extension_type, size_type >::RMQ

RMQ.

Given SA[k] and SA[k + 1] respectively induced from SA[i] and SA[j], the LCP-value of suf(SA[k]) and suf(SA[k + 1]) is determined by LCP[i + 1, j]. The response to the range-minimum query for LCP[i + 1, j] returns the minimum value among LCP[i + 1], .., LCP[j - 1], LCP[j].
