Detailed Description

A two-phase validator for suffix and LCP arrays verification.

Copyright (c) 2016, Sun Yat-sen University, All rights reserved

A Karp-Rabin method is employed to check SA_LMS & LCP_LMS. This method is also applied to the verification for SA and LCP, where an implementation can be found in validate3.h/validate3.cpp When finished checking SA_LMS & LCP_LMS, we validate SA & LCP following the idea of the induced-sorting principle and RMQ.

Note
The alphabet size of the input string is assumed to be constant.
