\begin{DoxyNote}{Note}
The alphaber size of the input string is assumed to be O(1) or O(n). 

This method requires multiple sequential scans for the input string. During each scan, all the fingerprints are computed by an iterative way. This has become the bottleneck to the system\textquotesingle{}s running time.
