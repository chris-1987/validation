\subsection{Detailed Description}
A two-\/phase validator for suffix and L\+CP arrays verification. 

Copyright (c) 2016, Sun Yat-\/sen University, All rights reserved

A Karp-\/\+Rabin method is employed to check S\+A\+\_\+\+L\+MS \& L\+C\+P\+\_\+\+L\+MS. This method is also applied to the verification for SA and L\+CP, where an implementation can be found in \hyperlink{validate3_8h}{validate3.\+h}/validate3.cpp When finished checking S\+A\+\_\+\+L\+MS \& L\+C\+P\+\_\+\+L\+MS, we validate SA \& L\+CP following the idea of the induced-\/sorting principle and R\+MQ.

\begin{DoxyNote}{Note}
The alphabet size of the input string is assumed to be constant.
