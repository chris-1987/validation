class async\+\_\+multi\+\_\+stream\+\_\+writer$<$ value\+\_\+type $>$}

multi-\/stream writer 

Main thread write data to multiple files at the same time. For each file, create a set of buffer and an I/O thread to write data to disk by aysnchronous I/O operations.
