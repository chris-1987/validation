\subsection{Detailed Description}
first method for validating suffix and L\+CP arrays 

Copyright (c) 2016, Sun Yat-\/sen University, All rights reserved

Given suffix and L\+CP arrays, validate their correctness using external memory sorts and Karp-\/\+Rabin Finger-\/printing function.

The alphabet of input string is assumed to be O(1) or O(n).
